# Stochastic Energy Management System (MDP)

## Objective
Optimize battery dispatch and grid trading in a renewable microgrid using Markov Decision Processes.

## Key Concepts
- Markov Chains
- Stochastic Dynamic Programming
- Time Series Forecasting
- Convex Optimization

## Tech Stack
- Python (NumPy, SciPy, CVXPY)
- r-stat
